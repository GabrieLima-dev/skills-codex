Skills Workflow Guide

Guia operacional para uso de skills, governança e gates obrigatórios de execução.

Fluxo obrigatório

  • project-readiness-gate deve rodar antes de qualquer execução operacional.
  • Tríade de stack obrigatória: backend + frontend + docs; sem isso, execução fica bloqueada.
  • Em alterações de skills/ ou AGENTS.md, atualizar obrigatoriamente o pacote: doc_skills/readme.md + doc_skills/skills-workflow-guide.html.
  • sdd-sprint-orquestracao organiza plano aprovado + comando explícito.
  • Para feature: sdd-pesquisa-prd -> sdd-prd-spec -> sdd-spec-code -> code-review -> sdd-validacao.

Governança mínima

  • Executar os quatro scripts de gate antes de publicar.
  • Registrar telemetria de skill: SKILL_START/STEP/HANDOFF/LOADING/END.
  • Em mudanças de skills/agente, atualizar obrigatoriamente doc_skills/readme.md e doc_skills/skills-workflow-guide.html.
  • Atualizar skills/CHANGELOG.md, skills/governance/CHANGELOG.md e skills/VERSION em mudanças estruturais.
