# Benchmark Visual (3 Modelos de Mercado)

## Modelo 1) Apple Editorial
- Link: [apple.com](https://www.apple.com/)
- Padrão: minimalismo premium, ritmo visual com alto respiro, foco em narrativa.
- Quando usar: reposicionamento visual elegante, hero forte e experiência aspiracional.

## Modelo 2) Stripe Product Docs
- Link: [docs.stripe.com](https://docs.stripe.com/)
- Padrão: estrutura limpa, alta legibilidade técnica, blocos funcionais bem segmentados.
- Quando usar: páginas operacionais/documentais com clareza de fluxo e execução.

## Modelo 3) Linear Product UI
- Link: [linear.app](https://linear.app/)
- Padrão: interface moderna, motion discreto, contraste refinado, microinterações objetivas.
- Quando usar: dashboards e fluxos com foco em produtividade e sensação de velocidade.

## Regra operacional obrigatória
- Em alteração grande de front, pedido de refazer todo o front ou demanda explícita de UX/UI: apresentar exatamente estes 3 modelos (ou equivalentes de mercado) com link clicável.
- Sempre incluir trade-offs curtos por modelo (prós e contras).
- Nunca implementar antes da escolha explícita do usuário.
