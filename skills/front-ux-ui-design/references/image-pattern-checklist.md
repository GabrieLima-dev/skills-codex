# Análise de Imagem de Referência (Frontend)

Quando o usuário enviar imagem de referência, extrair e reportar:

1. Estrutura
- Grid (colunas, gutters, alinhamento)
- Densidade (compacta, média, espaçada)

2. Tipografia
- Estilo (geométrica, humanista, serifada)
- Escala de títulos/corpo

3. Cor e contraste
- Paleta principal/secundária
- Relação de contraste em textos e CTAs

4. Componentes
- Botões, campos, cards, navegação
- Bordas (retas/arredondadas), elevação e sombras

5. Ambiência
- Fundo sólido, gradiente, blur, textura
- Nível de saturação e temperatura visual

6. Interação
- Estados hover/focus/active
- Padrão de transição e motion

## Regra obrigatória
- Antes de implementar, confirmar com o usuário: "é este padrão visual que você deseja replicar?"
